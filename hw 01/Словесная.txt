1. Открыть сайт https://www.booking.com/
2. Выберите валюту и язык.
3. В поле выбора населённого пункта ввести необходимый населённый пункт.
4. Если не находит введённый населённый пункт - поменяйте данные в пункте 3.
5. В поле выбора дат заезда-отъезда выбрать даты заезда/отъезда.
6. В поле выбора количества проживающих выбрать количество проживающих взрослях, количество детей, количество номеров.
7. Если путешествуете по работе - поставить метку "я путешествую по работе".
8. Нажать кнопку "Проверить цены".
9. Если нет свободных номеров - поменяйте при необходимости данные в пункте 6, 5, 3.
10. Выберите необходимые фильтры/очерёдность вывода номеров.
11. Выберите подходящее место размещения.
12. Если нет подходящего места размещения - поменяйте при необходимости данные в пункте 6, 5, 3.
13. Нажмите "Посмотреть наличие мест".
14. Выберите подходящий номер/несколько номеров.
15. Если номер занят - выберите другой номер, или нажмите "изменить параметры поиска" и поменяйте даты заезда/отъезда, количество путешествующих.
16. Если подходящий номер не найден - поменяйте при необходимости данные в пунктах 10, 9, 6, 5, 3.
17. Нажмите "я бронирую".
18. Заполните раздел "Введите свои данные".
19. Заполните при необходимости раздел "Задайте вопрос".
20. Нажмите "Далее: финальные данные".
21. Введите банковские реквизиты.
22. Нажмите "Забронировать".
